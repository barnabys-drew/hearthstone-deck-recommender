Starting/stopping the Electron app:

- Stop — from WSL:
cmd.exe /c "taskkill /im electron.exe /f"
- (Or just close it from Windows: Ctrl+Shift+9 hides it without stopping; Task Manager → electron.exe ends it.)
- Start — from WSL:
nohup cmd.exe /c "C:\Users\drewt\hearthstone-overlay\node_modules\electron\dist\electron.exe C:\Users\drewt\hearthstone-overlay" >/dev/null 2>&1 &
- Or from Windows PowerShell: cd C:\Users\drewt\hearthstone-overlay; npm start
